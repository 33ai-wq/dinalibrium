## What is Dinalibrium?

Dinalibrium uses **Dynamic Equilibrium** — a principle from physics — to analyze any asset.
One score tells you everything: the **DES Score (0–100)**.

| DES | Zone | Signal |
|-----|------|--------|
| 0–20 | Extreme Oversold | STRONG BUY |
| 20–42 | Under-Equilibrium | BUY |
| **42–58** | **⚖ Dinalibrium** | **MAINTAIN** |
| 58–80 | Over-Equilibrium | SELL |
| 80–100 | Extreme Overbought | STRONG SELL |
